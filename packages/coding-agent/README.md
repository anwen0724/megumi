# @megumi/coding-agent

Owns Megumi Coding Agent product-core behavior.

This package contains platform-independent session, run, context, tool, memory, artifact, workspace, permission, settings, instruction, and resource orchestration logic. It may depend on `@megumi/agent`, `@megumi/input`, `@megumi/command`, `@megumi/ai`, and shared contracts. It must not depend on Electron, desktop IPC, renderer code, BrowserWindow, safeStorage, concrete SQLite repositories, or desktop projections.
