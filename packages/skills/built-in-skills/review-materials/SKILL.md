# 综合学习资料进行复习

围绕用户当前复习目标综合已有资料，说明依据、冲突和缺口后形成可使用的结果。

## 确定复习目标

1. 识别用户要复习的范围、用途、输出形式和可用时间，例如考试提纲、知识结构、易错点、复习卡片或自测题。
2. 读取用户明确提供或授权使用的文字、图片和 Workspace 资料。不要默认扩大资料范围，也不要索取与当前任务无关的个人信息。
3. 只有目标或材料缺失到无法产生可靠结果时才追问；其余情况基于现有资料完成，并标明限制。

## 综合资料

- 提取各份资料中的核心概念、事实、例题、要求和容易混淆之处，合并重复内容但保留必要出处线索。
- 不预设课本、笔记或网络资料的绝对优先级。发现冲突时，明确列出冲突内容、相关来源和采用某个结论的依据。
- 区分资料直接支持的事实、根据资料作出的推断和仍然存在的不确定性；不要补写资料中不存在的内容来制造完整感。
- 按用户目标重新组织信息，而不是逐份材料机械摘要。需要时建立主题、前置关系、对比或时间顺序。
- 生成自测内容时，让问题覆盖整理出的重点，并按用户要求决定是否附答案。

## 交付结果

先给出与目标直接相关的复习成果，再简短说明资料缺口、冲突或需要用户核对的地方。使用用户当前语言。用户要求保存时遵循指定路径；未指定路径时使用当前 Workspace 根目录，并说明生成的文件。
